DIAGNOSTIC TROUBLE CODE (DTC) 0652 LEFT FRONT ISOLATION OPEN OR DRIVER SHORTED

Circuit Description

EBCM proceeds the self-test about the dump valve or the motor driver, if the EBCM senses the brake signal.

Diagnosis

If the EBCM proceeds the self-test about the dump valve or the motor driver and senses the fault, the EBCM may make the fault code.

Cause(s)

  • EBCM internal fault (Left front isolation voltage open or ground shorted).

Warning Lamp Operation

  • ABS warning lamp ON.
  • ABS warning lamp OFF if the ignition switch ON/OFF.

Fail Action

This is a critical operational fault. The ABS is disabled and the ABS warning lamp is turned on. The proportioning is operation.

DTC 0652 - Left Front Isolation Open or Driver Shorted

DIAGNOSTIC TROUBLE CODE (DTC) 2458 WHEEL SPEED SENSOR INTERMITTENT ERROR

Circuit Description

The toothed wheel speed ring generates a voltage pulse as if moves past the wheel speed sensor. Each tooth-gap-tooth series on the wheel generates the pulses. The Electronic Brake Control Module (EBCM) uses the frequency of these pulses to determine wheel speed. The voltage generated depends on the air gap between the wheel speed sensor and the toothed wheel, and on the wheel speed.

Diagnosis

During the driving, the serious air gap intantly between the wheel speed sensor and the toothed wheel happens.

Cause(s)

  • The serious air gap between the wheel speed sensor and the toothed wheel.

Warning Lamp Operation

  • When the fault happens, the warning lamp turns ON in a moment and OFF.

Fail Action

The ABS is operation and the proportioning is operation.

Diagnostic Aids

DTC 2458 is the external fault and ABS function OK. But the fault stores in the EBCM.
